TO DO LIST

This server provides GET, POST, PUT, AND DELETE Requests and Responses for the To Do List

/todo

Returns a list of all todos. However if an id is provided, it will only return the single requested id

id [number]: the id of the todo

Body Not Required

200 = Successful/OK
400 = Requested body not permitted
400 = Query Parameters do not meet requirements
400 = ID is not a number
400 = Todo with that ID not found

/todo

Adds a todo to the list. If a list parameter is provided it adds multiple to dos in one call

id[number](required): the id of the task to be updated to othe todolist

todo[string](required): the task to be added to the todolist list[array]:contains a list of todos

200 = Successful/OK
400 = Requested body not permitted
400 = Query Parameters do not meet requirements
400 = ID is not a number
400 = Todo with that ID not found

/todo/:id

Updates an existing to do and modifies its data

id[number](required): the id of the task to be updated to othe todolist

todo[string]:the task to be updated completed[boolean]: the status of the task

200 = Successful/OK
400 = Requested body not permitted
400 = Query Parameters do not meet requirements
400 = ID is not a number
400 = Todo with that ID not found

/todo/:id

Deletes an existing to do

id[number](required): the id of the task to be updated to othe todolist

Body Not Permitted

200 = Successful/OK
400 = Requested body not permitted
400 = Query Parameters do not meet requirements
400 = ID is not a number
400 = Todo with that ID not found
